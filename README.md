# Void Breaker Project


Void Breaker is a game in the same vibe as its predecessor "Void Piercer", aimed to be a roguelite fast fps game.

## Description

Void Breaker is a roguelite, Fast-Fps game heavely inspired by games like Ultrakill, Gunfire Reborn, Doom (1 & 2), Stress Level Zero universe & other universes.

the game takes place in a close future, a few years after the "Void Piercer" Project where the player controlls "Void Breaker", a high technology robot created by the same company that created Void Piercer. This time, the robot's goal is not to venture deep into the void, but to be the Apex predator of this unkwown place called the VoidWay. 

 ## Gameplay features

- fast paced FPS movement featuring
    -  wallruning
    -  walljumping
    -  dashing
    -  Sildes 

- Dynamic combat system
    - several ranged weapons 
    - mele / parry system
    - perks and upgradable weapons / character

- semi procedural generatino
    - different layer of several levels, with each layer having its own theme (W.I.P)
